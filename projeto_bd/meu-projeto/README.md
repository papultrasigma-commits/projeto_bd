## Como configurar a Base de Dados
1. Abre o pgAdmin 4 e cria uma base de dados vazia chamada `bricoloja`.
2. Clica com o botão direito na base de dados criada e seleciona **Query Tool**.
3. Abre o ficheiro `setup_bd.sql` que está na pasta `database/`.
4. Copia todo o código, cola na Query Tool e clica em **Execute** (Play). 
5. As tabelas serão criadas e os dados de teste inseridos automaticamente!
